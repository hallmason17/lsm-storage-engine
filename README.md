# LSM Storage Engine

Building an LSM-tree storage engine in C++ from scratch for learning.

## Goal
Understand how databases like RocksDB work by implementing the core concepts myself.

## Status

Currently implemented:
- [ ] Basic MemTable
- [ ] SSTable format
- [ ] Write-ahead log
- [ ] Compaction
- [ ] Bloom filters

## Building
```bash
mkdir build && cd build
cmake ..
make
```
